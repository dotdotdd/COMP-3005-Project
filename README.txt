Steps to compile

1. Ensure "url" is set to the user's database host, port, and database
2. Ensure "user" is set to the user's username
3. Ensure "password" is set to the user's password
4. Import and run the ddl and dml file to a database called project in pgAdmin
5. To start the program, open the program folder in IntelliJ and click the top right triangle "Run 'main'" button
6. The application will be presented as text on the from the terminal
7. Navigate through the application by selecting the option by typing in the numbers
8. Follow the instructions given at each step
